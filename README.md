# Aegis

Aegis is a CLI-based secure messaging platform that uses post-quantum cryptography to secure messages, with a zero-trust server architecture.

## Security Features

- Post-quantum cryptography
    - ML-KEM-1024 for the secret (AES-256-GCM) key exchange.
    - ML-DSA-87 to ensure that messages are not tampered with.
- Zero-trust server architecture (server cannot see messages)
- AES-256-GCM message encryption (quantum-resistant)
